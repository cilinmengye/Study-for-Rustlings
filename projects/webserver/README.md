通过线程池管理服务器端，线程池和工作线程的关系是生成者和消费者，生成和消费的内容为工作线程要执行的闭包
线程池接收来自客户端的请求，并将处理工作的闭包发送给工作线程，工作线程接收闭包并执行

远程连接服务器命令： 
ssh -p 40606 -L 7878:127.0.0.1:7878 username@remoteip
* -p 40606：SSH 端口
* -L 7878:127.0.0.1:7878：把本地的 7878 端口映射到远程服务器的 127.0.0.1:7878
* username@remoteip：登录远程的账号和主机
然后就可以通过本地浏览器输入http://127.0.0.1:7878/访问服务器监听的127.0.0.1:7878了

总共可以请求2种合法路径：
* http://localhost:7878/hello
* http://localhost:7878/sleep: 此是为了测试多线程
其余路径将会得到404.html
